# variance

## Firma

```python
variance(
    x: ArrayLike,
    w: ArrayLike | None = None,
    ddof: int | float = 0,
) -> float
```

`variance` calcola la varianza di un campione nel caso non pesato o, in alternativa, nel caso pesato con denominatore corretto da `ddof`.
