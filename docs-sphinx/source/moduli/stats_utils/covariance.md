# covariance

## Firma

```python
covariance(
    x: ArrayLike,
    y: ArrayLike,
    w: ArrayLike | None = None,
) -> float
```

`covariance` stima la covarianza tra due variabili numeriche, con supporto opzionale ai pesi e con validazione coerente degli input.
