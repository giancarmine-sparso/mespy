# standard_deviation

## Firma

```python
standard_deviation(
    x: ArrayLike,
    w: ArrayLike | None = None,
    ddof: int | float = 0,
) -> float
```

`standard_deviation` restituisce la deviazione standard come radice quadrata della varianza calcolata da [`variance`](variance.md).
